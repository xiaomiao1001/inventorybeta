# 郭四车行库存管理系统 - 经销商功能原型

## 项目概述

本原型为郭四车行库存管理系统的经销商功能模块，基于需求文档设计的高保真交互原型。

## 技术栈

- **HTML5** - 结构标记
- **Tailwind CSS** - 样式框架
- **FontAwesome** - 图标库
- **Unsplash** - 真实图片资源

## 设计规范

### 设备适配
- **目标设备**: iPhone 16 Pro Max
- **屏幕尺寸**: 430×932px
- **设计风格**: iOS原生 + 微信小程序

### 设计系统
- **主色调**: #0066cc (蓝色)
- **辅助色**: #ff6b6b (红色)、#28a745 (绿色)
- **字体**: -apple-system, BlinkMacSystemFont
- **圆角**: 12px (卡片)、24px (按钮)
- **阴影**: 0 2px 8px rgba(0,0,0,0.04)

## 页面结构

### 1. index.html - 原型总览
- 展示所有页面的iframe预览
- 手机框架模拟真实设备
- 设计说明和技术介绍

### 2. order.html - 经销商下单页面
- **功能**: 车辆配置、增加配置、费用计算、提车方式
- **特色**: 实时价格计算、级联选择、配置可视化
- **交互**: 多选配置、价格联动、表单验证
- **导航**: 底部双标签导航（下单/订单）

### 3. orders.html - 订单列表页面  
- **功能**: 订单展示、状态筛选、点击查看详情
- **特色**: 状态标签、简洁信息展示、统计数据
- **交互**: 状态筛选、卡片点击、快速操作
- **导航**: 底部双标签导航（下单/订单）

### 4. detail.html - 订单详情页面
- **功能**: 完整订单信息、状态流程、操作功能
- **特色**: 状态时间线、价格明细、信息分组
- **交互**: 状态跟踪、价格展开、操作确认
- **访问**: 从订单列表页面点击进入

### 5. empty.html - 空状态页面
- **功能**: 空状态引导、功能介绍、服务信息
- **特色**: 插图设计、功能卡片、联系方式
- **交互**: 引导下单、功能展示

## 关键特性

### 🎨 视觉设计
- **现代化界面**: 渐变色、卡片式布局、微交互
- **信息层级**: 清晰的视觉层次和信息组织
- **品牌一致性**: 统一的色彩和字体规范

### 📱 交互体验
- **简化导航**: 双标签底部导航（下单/订单）
- **状态反馈**: 悬停效果、加载状态、操作反馈
- **表单优化**: 智能校验、实时计算、错误提示
- **点击交互**: 订单卡片点击查看详情

### 💰 业务逻辑
- **价格体系**: 高级/低级批发商差异化定价
- **配置管理**: 车型级联选择、配置价格计算
- **订单流程**: 完整的订单状态管理

### 🚀 技术实现
- **响应式设计**: 适配不同屏幕尺寸
- **模块化结构**: 独立页面文件，便于维护
- **真实数据**: 使用真实图片和业务数据

- **批量提交**：

  - 显示订单详情和总金额确认

- **批量删除**：快速清理不需要的草稿订单
- **状态管理**：提交后自动更新订单状态，隐藏复选框
