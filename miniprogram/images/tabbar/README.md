# TabBar 图标说明

## 图标列表

基于 TDesign 设计规范的底部导航栏图标：

### 1. 首页 (Home)
- **图标名称**: `home`
- **设计**: 简洁的房子图标
- **颜色**: 未选中 `#999999`，选中 `#007AFF`

### 2. 库存 (Shop)
- **图标名称**: `shop`
- **设计**: 购物车/店铺图标
- **颜色**: 未选中 `#999999`，选中 `#007AFF`

### 3. 出库 (Order)
- **图标名称**: `order`
- **设计**: 订单列表图标
- **颜色**: 未选中 `#999999`，选中 `#007AFF`

### 4. 统计 (Chart)
- **图标名称**: `chart`
- **设计**: 柱状图表图标
- **颜色**: 未选中 `#999999`，选中 `#007AFF`

## 使用方法

1. 将 `icons-template.svg` 文件中的每个图标导出为 PNG 格式
2. 尺寸建议：
   - 普通状态：24x24px (最小)
   - 高清状态：48x48px (推荐)
   - 超高清状态：72x72px (可选)

## 文件命名规范

- 未选中状态：`[icon-name].png`
- 选中状态：`[icon-name]-active.png`

例如：
- `home.png`, `home-active.png`
- `shop.png`, `shop-active.png`
- `order.png`, `order-active.png`
- `chart.png`, `chart-active.png`

## TDesign 图标特点

- 线条简洁，视觉统一
- 符合现代扁平化设计趋势
- 适合小程序底部导航栏使用
- 支持明暗两种状态切换

## 注意事项

- 确保图标在不同屏幕密度下清晰显示
- 保持图标的一致性和可识别性
- 建议使用透明背景的PNG格式
